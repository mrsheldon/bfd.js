# Getting Started
Simply Run `npm install bfd.js`

# Example
```javascript
onst BFD = require('bfd.js');
const BFDClient = new BFD('Bot ID', 'Bot Token');

BFDClient.update(client.guilds.size)

console.log("Server Count Updated")
```
